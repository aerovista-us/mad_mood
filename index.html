Album concept

A portrait record built on taste, memory, standards, family legacy, and the dream of quiet peace — not false hardship mythology.

Overall arc

The record opens with identity, moves through memory and refinement, reveals its emotional center in pride for good kids, pushes back against polished counterfeit culture, then closes by drifting from abstraction into self-summary and finally into the earned exhale of Spain.

Design cues

Moody analog art-rock energy, vintage elegance, restrained luxury, dark-glass presentation, smooth motion, and lock-screen-ready playback with an install flow that feels native on mobile.

About EchoVerse | AeroVista

EchoVerse is the album and sound-design lane inside AeroVista — built for immersive, story-first listening experiences that feel polished on mobile, lightweight on the web, and expressive enough to carry an album like a digital liner note, player, and presentation in one place.

About this album

Mad Mood Music is a portrait of a man defined by taste, restraint, memory, standards, good kids, and a dream of retiring in Spain. The project works best as elegance and honesty rather than fake struggle.

Playback behavior

  • Auto-plays the next track at the end of each song.
  • Saves your current track and timestamp locally for return visits.
  • Uses Media Session for lock-screen metadata and transport controls when supported.
  • Designed so playback can continue during screen-off or background use whenever the device/browser allows it.

GitHub Pages ready

This build uses a static app shell, local asset paths, a manifest, and a service worker so it can deploy cleanly to GitHub Pages with no backend required.
